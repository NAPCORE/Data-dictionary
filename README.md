## About this Repository
This repo originally hosted the first iterations of the **NAPCORE Data Dictionary**, a collaborative effort to harmonize terminology, definitions, and metadata across European ITS and mobility data catalogues.

It served as a starting point for:
- Collecting draft definitions and concepts
- Testing approaches for open data catalogue alignment
- Sharing early documentation and examples
